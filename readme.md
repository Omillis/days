# days [![Build Status](https://travis-ci.org/saadshahd/days.svg?branch=master)](https://travis-ci.org/saadshahd/days)

> Get milliseconds from days


## Install

```
$ npm install --save @saadshahd/days
```


## Usage

```js
const days = require('@saadshahd/days');

days(1); // 86400000
```
